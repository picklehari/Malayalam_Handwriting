README

The folder includes mat file contains 200 sample handwritten Malayalam character images  
With the help of reshape function in matlab, vector form of images can be converted to the 32*32 image format.


MalayalamHandwritten.mat : Contains the vector form of character images with each column represents the character image.
The first element in each column represents the character class number and the rest 1024 values represents the vector form of character images.  

For the time being, we can provide only sample images.
